## Game QA Testing Project

## Bug Report #001

### Title:
Item disappears from inventory after restarting the game

### Bug ID:
GAME-001

### Severity:
Major

### Priority:
High

### Environment:
- Platform: PC
- OS: Windows
- Testing Type: Manual Testing

### Preconditions:
- Player has at least one item in inventory

### Steps to Reproduce:

1. Launch the game
2. Collect an item
3. Open inventory
4. Save the game
5. Restart the game
6. Open inventory again

### Actual Result:

The collected item is missing from the inventory.

### Expected Result:

The item should remain saved and available after restarting the game.

### Status:

Open
